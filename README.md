# Gato 4x4 Inteligente - Minimax con Poda Alfa-Beta

Este repositorio contiene la implementación de un agente inteligente capaz de jugar de manera óptima al juego del Gato (Tres en raya) en un tablero expandido de 4x4. 

El proyecto fue desarrollado como parte de la Práctica 3 de la materia de Inteligencia Artificial.

## 🚀 Características Principales
* **Algoritmos implementados:** Minimax Clásico vs. Minimax con Poda Alfa-Beta.
* **Límite de profundidad:** 4 niveles de búsqueda predictiva.
* **Métricas en tiempo real:** Comparativa visual de los nodos explorados por ambos algoritmos tras cada turno de la máquina.
* **Función Heurística Personalizada:** Evalúa el control del centro del tablero, prioriza la creación de líneas abiertas y ejecuta bloqueos de emergencia cuando el oponente alinea 3 fichas.

## 🛠️ Tecnologías Utilizadas
El proyecto está construido íntegramente con tecnologías web nativas, sin dependencias externas:
* **HTML5:** Estructura semántica.
* **CSS3:** Diseño responsivo, Grid Layout y variables de color.
* **JavaScript (ES6+):** Lógica del motor de juego y algoritmos de IA.

## ⚙️ Cómo ejecutar el proyecto
1. Clona este repositorio o descarga los archivos en formato `.zip`.
2. Extrae los archivos en una carpeta local.
3. Haz doble clic sobre el archivo `index.html` para abrirlo en tu navegador web de preferencia.
4. ¡Juega contra la IA (empiezas tú con la 'X')!
